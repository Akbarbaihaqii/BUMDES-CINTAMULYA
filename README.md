# Install
```
composer i
```
```
npm i
```

# Settings
 Ganti DB_HOST,DB_PORT,DB_DATABASE,DB_USERNAME,DB_PASSWORD sesuai dengan database yang akan dipakai
# Migrasi
```
composer migrate:fresh --seed
```
# Menambahkan akun admin
```
php artisan make:filament-user
```
# Menjalankan projek
```
php artisan serve
```
